Web Development

I have a good understanding of web technologies. I've been using HTML, CSS, and JavaScript to create project websites. I'm also very familiar with responsive programming for both desktop and mobile devices. On top of that, I'm also able to implement asynchronous elements on the website such as fetching data from external APIs. Apart from plain HTML and CSS, I create some of my projects in React framework. On the backend side of web programming, I primarily use NodeJS for creating web servers and to handle user requests.

Mobile Development

I'm a novice when it comes to mobile programming, however, I have some familiarity with technologies such as React Native and Flutter. I'm able to create basic mobile apps and deploy them to physical devices.
